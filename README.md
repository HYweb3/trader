## ✅ 一、交易核心逻辑（五步骤）

1️⃣ **定趋势** → 顺应趋势  
2️⃣ **找关键** → 支撑与阻力  
3️⃣ **看信号** → 入场时机  
4️⃣ **算盈亏** → 盈亏比与风险回报  
5️⃣ **下订单** → 仓位与风控管理  

---

## ✅ 二、交易理念与核心思维

- 重趋势，轻价格  
- 高位天量要出货  
- 低位天量要买进  
- 主升浪靠拿住  
- 均线 + 主升浪战法：  
  - 低位放量 → 买入  
  - 等待主升浪  
  - 分批卖出  

---

## ✅ 三、趋势判断与方向过滤（2.1）

### 📌 趋势具有惯性  
除非有明确转变信号，否则不轻易猜顶/猜底。

**趋势反转识别：**
- 跌破趋势线 + 回测不站回
- 双顶 / 双底
- 头肩顶 / 头肩底
- 均线发散方向
- 多周期共振确认

---

### 📉 三大行情结构  
- 上涨 → 回调做多  
- 下跌 → 反弹做空  
- 震荡 → 区间高空低多

---

### 📊 恐慌指数参考  
- 指数 < 30 → 谨慎开空  
- 指数 > 70 → 谨慎开多  

---

### 📍 价格与均线判断（MA55 强弱）  
- 价格在 MA55 上方 → 做多为主  
- 价格在 MA55 下方 → 做空为主

🟢 金叉 → 强势，只做多  
🔴 死叉 → 弱势，只做空  

---

### 📏 顺大逆小原则

**以大级别（日线 / 周线）为主方向：**

- ✅ 如果大级别强势（上涨趋势）  
  → 小周期只找多头机会  
  → 逆势空单仅做短线 & 小仓  

- ✅ 如果大级别弱势（下跌趋势）  
  → 小周期只找做空机会  
  → 抄底多单 ≤ 10% 仓位  

👉 核心：**大顺势，小逆势，不反大周期。**

---

## ✅ 四、找关键位（2.2）

### 🔹 区间震荡关键位  
- 上沿压力：多次触顶回落  
- 下沿支撑：多次触底反弹  
- 反复触及，有明显买卖反应

### 🔹 趋势回调关键位  
- 前高/前低结构位  
- MA15 / MA55 动态支撑压力  
- 趋势线回踩 /反抽点位  
- 前波回调黄金分割位

## ✅ 五、入场信号分级体系（强度+执行标准）

为保证执行一致性，将所有入场触发条件分为 4 个等级：

### 🟢 S级信号（可独立开仓 / 主仓位信号）
- 趋势首次回调 + MA15/MA55 支撑/压力反转K线
- 大周期顺势 + 前高/前低关键位反转
- 区间突破后放量确认 + 收盘站稳
- 主趋势中继洗盘后放量反转（强势趋势中）

✅ 允许：40%–50% 仓位  
✅ 无需等待辅助信号  
✅ 可分批加仓

---

### 🔵 A级信号（需趋势确认 / 中仓位信号）
- 放量下跌 + RSI ≤30 + 金叉趋势中
- 回踩趋势线 + 长影线反转
- 止跌形态 + 均线共振（MA15/55）

✅ 允许：20%–30% 仓位  
✅ 必须顺大级别方向  
✅ 允许等待确认K线

---

### 🟡 B级信号（试探仓 / 小仓布局）
- RSI 二次超卖反弹（逆势底抄）
- 小周期假突破 + 背离
- 区间下沿/上沿博弈无放量

✅ 允许：5%–10% 仓位  
✅ 仅短线打法  
✅ 必须设置止损 ≤2%

---

### 🔴 禁止单独开仓信号（仅作辅助参考）
- 单一K线形态（针、吞没、孕线等）
- 无趋势背景的RSI/背离类信号
- 无量结构博弈
- 猜顶/摸底行为
- 超短周期噪音信号（5-15m孤立信号）

❌ 禁止单独进场  
❌ 禁止放大仓位  
❌ 仅作为观察信号

---

### ✅ 信号共振原则（叠加加分）

| 条件组合 | 强度 | 行动 |
|----------|------|------|
| S级信号 + 顺大趋势 | ★★★★★ | 主仓进场 |
| A级信号 + 关键结构位 | ★★★★☆ | 中仓进场 |
| B级信号 + 大级别趋势一致 | ★★★☆☆ | 小仓试单 |
| 单信号 + 无趋势支撑 | ★★☆☆☆ | 观望等待 |
| 逆势单 + 信号不足 | ★☆☆☆☆ | 禁止进场 |

## ✅ 六、多周期执行框架（方向、结构、入场三分法）

为了避免信号混乱、逆势操作、提前/滞后进场，采用「三周期联动」模型进行执行：

### 🟥 ① 大周期（定义方向 / 不用于操作）
参考：周线 / 日线  
- 判断主趋势  
- 过滤逆向交易  
- 辨别行情阶段（主升浪/回调浪/末端）  
✅ 不直接下单，仅定方向

---

### 🟧 ② 中周期（执行节奏 / 找结构位）
参考：4H / 1H  
- 识别趋势线、区间、前高/前低  
- 找关键回调/反抽位  
✅ 用于布局，不直接开仓

---

### 🟩 ③ 小周期（执行信号 / 精准开仓）
参考：15m / 30m  
- 捕捉反转K、突破、量能  
- 精准进场 & 止损  
✅ 真正执行开仓的周期

---

### ✅ 三周期联动口诀
```
大周期定方向
中周期找位置
小周期觅信号
趋势不明不重仓
结构不清不出手
信号不足不开单
```

## ✅ 七、仓位与风险管理系统（执行核心）

### ① 总资金风险控制
- 单笔亏损 ≤2%  
- 单日亏损 ≤5%  
- 连续止损3次 → 停止交易  
- 回撤>10% → 降级执行模式  

---

### ② 仓位分配模型

| 信号等级 | 顺势？ | 允许仓位 |
|----------|-------|----------|
| 🟢 S级   | ✅ 必须 | 40–50% |
| 🔵 A级   | ✅     | 20–30% |
| 🟡 B级   | ⚠️     | 5–10% |
| 🔴 禁止  | ❌     | 0% |

---

### ③ 加减仓规则
- ✅ 浮盈 ≥4% 且趋势延续 → 分批加仓  
- ❌ 不浮亏加仓  
- ✅ 亏损到止损位 → 减半/清仓  

---

### ④ 连续亏损机制
```
1次止损 → 降仓10%
2次止损 → 暂停周期
3次止损 → 停止当日交易
```

---

### ⑤ 盈亏比要求
- 最低 1:2  
- 止盈：1R/2R分批，剩余用EMA止盈  

---

- [ ] 止损是否明确？

# ✅ 二、进场规则（信号来源）

## 🔹 首次回调策略  
**趋势首次回调 → 顺势进场**

- ✅ **强势环境：**
  - 回踩 MA15/55 或趋势线支撑  
  - 出现反转K线信号 → **做多**

- ✅ **弱势环境：**
  - 反抽 MA15/55 或压力位  
  - 出现反转K线信号 → **做空**

---

## 🔸 区间震荡策略

### 🔁 区间内操作（大概率）  
- 区间顶部 → 做空  
- 区间底部 → 做多  

### 🚀 区间突破（小概率）  
**前提条件：**
- 横盘 ≥ 20 根K线  
- 突破方向收盘确认  
- 成交量放大 ≥ 1.5 倍  

**方向：**
- 突破上沿 → 多单  
- 跌破下沿 → 空单  

**建仓方式：**
- 🟡 保守型：突破后回踩确认 → 分批建仓  
- 🟢 稳健型：突破确认 → 建仓 60%，回踩补仓  
- 🔴 激进型：小仓试单，每涨 5–8% 加仓；跌回 8–10% 清仓

---

## 🟢 做多时机

### ✅ 趋势回调进场  
- 趋势首次回调 → 顺势进多  
- 回踩 MA15/55 或趋势线 + 反转K线 → 多单

### ✅ 放量下跌 + RSI 低位  
**前提：趋势强势（金叉）**  
- 放量下跌  
- RSI ≤ 30–35  
- 出现长下影/背离 → 多单（洗盘捕捉）

### ✅ 下跌趋势中 RSI 双次超卖  
- 小时图 RSI <30 两次且未创新低  
- 小仓抄底 → 逆势单 ≤10%

---

## 🔴 做空时机

### ✅ 趋势回调顺势空  
- 趋势首次回调，顺大势进空

### ✅ 弱势环境（死叉）下操作  
- 价格反抽至 MA15/55 或前高压力位  
- 放量滞涨/反转K线 → 空单

### ✅ 区间跌破  
- 突破长时间横盘区间
- 放量确认下破  
- 反弹确认跌破 → 做空

### ✅ RSI 顶背离放量回落  
- 价格创新高、RSI 未创新高  
- 放量冲高回落 → 高位空单

---

# 🎯 三、出场规则

- ⛔ **止损：**
  - 入场点回撤 2–3%  
  - 或 ATR ×1.5

- ✅ **止盈：**
  - 1R / 2R 分批止盈  
  - 剩余仓位用移动止损（10EMA / 20EMA）

# 📊 四、仓位管理

## ✅ 做多仓位
- 趋势顺势单：40–50%  
- 区间突破单：20–30%  
- 放量下跌 + RSI 低位：20–30%（仅金叉时）  
- RSI 抄底单：≤10%

## 🔻 做空仓位
- 大级别弱势 → 30–40%  
- 大级别强势下仅短空 → ≤10%

 **区间突破 → 顺势做单【小概率】**  
 - 横盘 ≥20 根K  
 - 突破方向收盘确认，成交量放大 ≥1.5 倍  
 - 突破上沿 → 多单；跌破下沿 → 空单
 - **策略：**
 - 保守型：突破后回踩确认支撑/阻力 → 分批建仓  
 - 稳健型：突破确认 → 先建 60%，再等回踩补仓  
 - 激进型：小仓试单，每上涨 5–8% 加仓，跌回 8–10% 清仓
     
## 🟢**做多时机**       

**趋势首次回调 → 顺势进场**  
   - 强势：回踩 MA15/55 或趋势线支撑，反转K线信号 → 多单
   -  
**放量下跌 + RSI 低位 → 多单（洗盘捕捉）**  
   - 前提：趋势强势（金叉）  
   - 放量下跌，RSI ≤30–35，并出现长下影/背离 → 进多  

**下跌趋势中 RSI 两次超卖 → 短线抄底**  
   - 小时图 二次 RSI <30 且未创新低 → 小仓多单（逆势单，≤10%）  


## 🔴**做空时机**  

**趋势首次回调 → 顺大势进场**  
   - 弱势：反弹测试 MA15/55 或趋势线支撑，反转K线信号 
   - 
   - **弱势环境（死叉）**下：  
     - 价格反抽至 MA15/55 或前高压力位，出现放量滞涨/反转K线 → 空单  
     - 区间下破 → 放量确认后做空  反弹确认跌破
     - RSI 背离（价格新高，RSI 未创新高）+ 放量冲高回落 → 高位空单  

---

## 三、出场规则  
- **止损**：入场点 2–3% 或 ATR×1.5  
- **止盈**：1R/2R 分批止盈，剩余仓位用移动止损（10EMA/20EMA）  

---

## 四、仓位管理  
- 趋势顺势单：40–50%  
- 区间突破单：20–30%  
- 放量下跌RSI低位单：20–30%（仅金叉时）  
- RSI 抄底单：≤10%  
- 做空单：  
  - 若大级别弱势 → 30–40%  
  - 若大级别强势，仅短线逆势 → ≤10%  

---

📌 **总结**  
- 方向过滤：顺大逆小，主趋势优先。  
- 做空逻辑：死叉下优先反抽空、区间下破空、高位背离空。  
- 多空都有进场策略，但仓位分配不同，保证顺势大仓，逆势小仓。
